# Online-Quiz-App

### Description

    This web application provides the services which can be used to make your own tests and share them with others.

## Stack

#### Frontend:

- React
- Redux
- Sass
- Redux-thunk

#### Backend:

- Spring + SpringBoot
- PostgreSQL database
- Spring JPA + Hibernate
  It has a JWT bases authentication system.
